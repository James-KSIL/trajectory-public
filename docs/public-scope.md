# Public Scope

This repository is a curated public surface for Trajectory.

## What Is Intentionally Not Included

- proprietary business logic
- backend services and operational internals
- full source implementation and complete models

## Why IP Is Protected

The production system contains domain-specific logic and implementation details that are not disclosed in a public portfolio context.

## What This Repository Demonstrates

- Flutter/Dart product execution
- deterministic system architecture thinking
- practical UI and workflow delivery shown through real screenshots
