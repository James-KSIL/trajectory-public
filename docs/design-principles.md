# Design Principles

## Determinism Over Heuristics

Core outcomes are generated from explicit rules, not probabilistic estimation.

## Explicit State Transitions

State changes are represented as deliberate transitions with traceable inputs and outputs.

## Auditability

System behavior is structured to be inspectable and explainable at each stage.

## Offline-First Operation

The MVP prioritizes local persistence and on-device continuity over cloud dependency.

## Separation of Calculation vs Presentation

Calculation pathways and UI rendering concerns are separated to maintain clarity and reliability.
